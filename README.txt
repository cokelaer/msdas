This packagse gathers data and scripts used in the context of the CombiMS
project.


The layout is python but the content could be made of R or python scripts.

The share/data directory contains a PKN and 7 MIDAS files.

The src/msdas directory contains R scripts that are copies of 9-7 functions
from  CNORode package that required some changes to implement a new tye of
optimisation.

The src/msdas contains a script called main.R . It is commented and should
work out of the box.
